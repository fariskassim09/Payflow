# 🚀 Quick Start - Build Payflow APK/IPA with EAS

## 3 Simple Steps:

### Step 1: Create Free Expo Account
- Go to https://expo.dev
- Click "Sign Up"
- Verify email

### Step 2: Go to EAS Build
- Visit https://expo.dev/eas
- Login with your account
- Click "New Build"

### Step 3: Choose Your Platform

**For Android APK:**
1. Select "Android"
2. Choose "APK"
3. Click "Build"
4. Wait 5-15 minutes
5. Download to phone
6. Tap APK file to install

**For iOS IPA:**
1. Select "iOS"
2. Choose "Internal Distribution"
3. Click "Build"
4. Wait 10-20 minutes
5. Get TestFlight link
6. Install on iPhone

---

## That's It! 🎉

Your Payflow app will be built in the cloud and ready to download.
