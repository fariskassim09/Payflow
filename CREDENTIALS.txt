HOW TO USE
==========
1. Download payflow-upload-keystore.jks from GitHub
2. Go to https://expo.dev/eas
3. Select Payflow project
4. Go to Credentials → Android
5. Click "Upload Keystore"
6. Select payflow-upload-keystore.jks
7. Enter all passwords and key alias above
8. Click Upload
9. Build APK!

- Don't share with anyone
- Keep backup of keystore file
- Valid for 30 years
